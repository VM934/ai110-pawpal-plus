# PawPal+ (Module 2 Project)

PawPal+ is a pet care planning app that turns a list of owner tasks into a daily schedule. The backend is intentionally separated from the Streamlit UI so the scheduling rules can be tested without launching the app.

## What It Does

- Represents owners, pets, tasks, scheduled tasks, skipped tasks, and daily plans with Python classes.
- Sorts flexible tasks by priority, preferred time, duration, and title.
- Places fixed-time tasks first and detects overlapping fixed tasks.
- Handles recurring tasks: once, daily, weekdays, weekends, and weekly by selected weekday.
- Applies owner availability and daily care-minute limits.
- Explains why each scheduled task was placed and why skipped tasks were not scheduled.

## Streamlit App

```bash

```

The app lets a user enter owner availability, pet information, task details, priority, recurrence, preferred/fixed times, and weekly days. It then displays scheduled tasks, skipped tasks, care-minute totals, and the text explanation generated by the backend.

## Smarter Scheduling

| Feature | Method(s) | Notes |
|---------|-----------|-------|
| Task sorting | `PawPalScheduler._task_sort_key()` | Flexible tasks sort by priority first, then preferred time, duration, and title for deterministic results. |
| Filtering | `CareTask.occurs_on()` | Recurrence rules decide whether a task belongs in the selected day. Non-due tasks are skipped with a reason. |
| Conflict handling | `detect_conflicts()`, `_schedule_fixed()`, `_has_overlap()` | Fixed tasks are placed first; overlapping fixed tasks are skipped so the final plan does not double-book the owner. |
| Time windows | `_open_windows()`, `_first_available_start()` | Flexible tasks are placed into gaps inside owner availability. |
| Daily limit | `generate_plan()` | A task is skipped when it would exceed the owner’s daily care-minute limit. |
| Explanations | `ScheduledTask.reason`, `SkippedTask.reason`, `DailyPlan.summary()` | The app surfaces why each decision happened. |

## Demo Walkthrough

1. Enter the owner’s available start/end time and optional daily care-minute limit.
2. Enter pet name, species, age, and the date to plan.
3. Add care tasks with duration, priority, recurrence, and optional preferred/fixed times.
4. Click `Generate plan`.
5. Review scheduled tasks, skipped tasks, total care minutes, and the plain-text explanation.
